# Academic and Training References

This appendix collects external resources that help contributors improve
engineering practices used in scientific software projects.

## Recommended resources

- [The Carpentries](https://carpentries.org/): practical training material for reproducible computational work.
- [The Carpentries on GitHub](https://github.com/carpentries): open lesson repositories and examples.
- [Practical VCS Guide](https://practicalseries.com/1002-vcs/): version-control fundamentals useful for reviewable contributions.

## How to use this list

Use these references to onboard new contributors and align team-level software
engineering habits. This page is complementary to project-specific rules in
[Contributing Workflow](contributing-workflow.md) and
[Contributor Checklist](contributor-checklist.md).
