# Installation

PyUnitWizard can be installed with conda or pip.

## Basic installation

```bash
conda install -c uibcdf pyunitwizard
```

or:

```bash
pip install pyunitwizard
```

## Install from source

```bash
git clone https://github.com/uibcdf/pyunitwizard.git
cd pyunitwizard
pip install -e .
```
